# Online Course Platform

This is a Spring Boot REST API project developed to manage an online course platform.

## Modules
- Instructor
- Course
- Student
- Enrollment
- Lesson

## Features
- Instructors can create courses
- Students can enroll in courses
- Courses contain lessons
- Enrollment tracking and status
- Analytics endpoints (most enrolled course, courses with no enrollments)

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Database
- In-memory H2 database for development and testing

## API Testing
- APIs tested using Postman

## Project Type
- Backend REST API
